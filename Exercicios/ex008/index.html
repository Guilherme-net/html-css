Vamos Aprender Texto Subescrito

Basicamente, ele joga a palavra que você quer para CIMA. Utilizando a Tag SUP
Podemos Utilizar em alguns casos como: x20 + 72.

Para Finalizar, Vamos aprender agora texto Subscrito.

Como vimos no texto, o SubESCRITO joga pra cima, já o suBSCRITO joga pra baixo. Usamos a Tag sub para isso.

Cuidado, As Tags são parecidas mas não confunda. SUP é pra Cima, SUB é para Baixo.

Um exemplo onde podemos utilizar isso é: H2O.
